## Design System

- **Color palette:** Amber primary (`hsl(36 100% 50%)`) on deep navy background (`hsl(224 71% 4%)`)
- **Fonts:** DM Sans (display) + JetBrains Mono (code/numbers)
- **Aesthetic:** Linear/Vercel-inspired. Dark-first. No purple gradients.
- **Match scores:** Green ‚â•80, Amber 60‚Äì79, Red <60 ‚Äî via CSS custom properties
- **Accessibility:** `prefers-reduced-motion` kills all animations. `touch-action: manipulation` on buttons. `aria-label` on all icon buttons. Skip link. Semantic landmarks.
- **Typography:** `tabular-nums` on numeric displays. Unicode `‚Ä¶` everywhere. `text-balance` on headlines.
